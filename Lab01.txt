1. Change Directory: cd [folder]
2. Create a Directory: mkdir [dir]
3. Rename a Directory: mv [file] [new filename]
4. List the contents of a directory:  ls
5. Move a file to a new location: mv [file] [new filename]
6. Copy a file to a new location: cp [file] [dir]
7. Search for a string in a file: grep [search_pattern] [file]
8. Access the shell of a remote machine: ssh [address] (e.g. ssh root@example.com)
9. Remove a directory folder: rm -R [dir]
